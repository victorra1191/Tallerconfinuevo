## 🎨 CONSEJOS DE CALIDAD

### Para PROMOCIONES (más importantes):
- **Usa fotos reales de tu taller** trabajando en esos servicios
- **Buena iluminación** y alta resolución
- **Muestra mecánicos en acción** para generar confianza
- **Evita imágenes pixeladas** o de baja calidad

### Para PRODUCTOS:
- **Fondo blanco o transparente** para mejor presentación
- **Producto centrado** y bien iluminado
- **Sin marcas de agua** ni logos ajenos

### Para LOGO:
- **Fondo transparente** para que se adapte al header
- **Colores que combinen** con los colores del sitio (#D71920 y #004A9F)
- **Legible en tamaño pequeño**

## 🔄 ¿CÓMO VERIFICAR QUE FUNCIONAN?
